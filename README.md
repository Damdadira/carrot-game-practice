## 🕹 게임 규칙
- 제한 시간은 **5초**입니다.
- 5초 안에 **당근 5개**를 클릭하시면 **승리**하게 됩니다.
- **벌레**를 클릭하시면 즉시 **패배**하게 됩니다.
- **제한 시간** 안에 당근을 모두 클릭하지 못해도 **패배**하게 됩니다.
- 게임을 **다시 시작**할 때마다 **당근과 벌레의 위치는 무작위**로 배치됩니다.
